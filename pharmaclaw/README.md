# PharmaClaw Landing Page

## Deploy (5 mins)
1. **Netlify/Vercel**: Drag `index.html` to netlify.com/drop or vercel.com/new.
2. **Point DNS**: Set pharmaclaw.com A/CNAME to provider IP/URL.
3. **Custom domain**: Free on both.

## Customize
- Update ClawHub links (e.g., `/skills/chemistry-query`).
- Add Stripe: Replace Pro CTA with real checkout.
- Analytics: Google/Umami script in `&lt;head&gt;`.
- Privacy/Terms: Add `privacy.html`/`terms.html`.

## Tech
- Self-contained HTML/CSS/JS.
- Responsive, mobile-first.
- PWA-ready (add manifest later).
